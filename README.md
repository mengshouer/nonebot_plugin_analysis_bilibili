## 使用方式
私聊或群聊发送bilibili的小程序/链接

## 安装
pip install nonebot_plugin_analysis_bilibili

```
pip install --upgrade nonebot_plugin_analysis_bilibili
```
在 Nonebot2 入口文件（例如 bot.py ）增加：
``` python
nonebot.load_plugin("nonebot_plugin_analysis_bilibili")
```
